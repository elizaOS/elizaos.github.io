# elizaos-plugins/plugin-knowledge Daily Update (Nov 10, 2025)
## OVERVIEW 
Today, the plugin-knowledge repository saw a significant feature addition with the migration from pdfjs-dist to unpdf for universal PDF text extraction, streamlining PDF handling across various environments.

## KEY TECHNICAL DEVELOPMENTS

### Enhanced PDF Text Extraction
A major update replaced `pdfjs-dist` with `unpdf` for PDF text extraction [#46](https://github.com/elizaos-plugins/plugin-knowledge/pull/46). This change:
- Enables PDF text extraction in browser, serverless, and Node.js environments.
- Eliminates complex worker configurations.
- Reduces the dependency size from over 700KB to approximately 50KB.
