# elizaos-plugins/plugin-knowledge Daily Update (Nov 10, 2025)
## OVERVIEW 
Today, the `plugin-knowledge` repository saw a significant feature addition with the migration to `unpdf` for PDF text extraction, greatly simplifying setup and reducing dependencies. An issue regarding Web UI disabling was also opened and quickly closed.

## KEY TECHNICAL DEVELOPMENTS

### Enhanced PDF Text Extraction
A new feature was implemented to replace `pdfjs-dist` with `unpdf` for PDF text extraction. This change significantly reduces the library size and eliminates complex worker configurations, enabling seamless PDF processing across various environments. ([#46](https://github.com/elizaos-plugins/plugin-knowledge/pull/46))
